### 🚀 Proyectos Destacados
1. **[Recomendador de Coches con Machine Learning](https://github.com/sdahbaniiesruizgijon/Formulario_IA.git)** Un sistema inteligente en **Python** que analiza el perfil socioeconómico de un usuario (ingresos, presupuesto, género) y predice el modelo de coche ideal, mostrando el porcentaje de acierto del modelo.
   * *Tech Stack:* Python, PHP, Docker.
2. **[Sistema Rent a Car - Trabajo de Fin de Grado (TFG)](https://github.com/aalvarezruizgijon/Proyecto-Final.git)** Una plataforma web completa para la gestión y reserva de vehículos de alquiler. Desarrollada con **PHP nativo** bajo una arquitectura estricta de **Modelo-Vista-Controlador (MVC)**, destaca por reflejar fielmente las reglas de negocio, la gestión de inventario y el flujo de reservas de una empresa real.
   * *Tech Stack:* PHP, MVC, HTML5, CSS3, Bootstrap, MySQL.
   * * 🔗 **[Ver Demo en Vivo](https://ruix.iesruizgijon.es/sedahbani/drivo/Controller/)**
3. **[NutriSport - Plataforma de Nutrición y Recetas](https://github.com/sdahbaniiesruizgijon/ProyectoAngular.git)** 🥗 
   Una aplicación web de nutrición desplegada en vivo con un panel privado (Dashboard) para usuarios registrados. Funciona como un gestor de recetas inteligente que consume una **API externa** de alimentos para calcular automáticamente las kilocalorías y macronutrientes en tiempo real, aplicando una regla de negocio estricta que limita las recetas a un máximo de 2000 kcal.
   * *Tech Stack:* Angular, TypeScript, API REST, HTML5, CSS3, Bootstrap.
   * 🔗 **[Ver Demo en Vivo](https://ruix.iesruizgijon.es/sedahbani/ProyectoAngular/frontend/)**
   * 
4. **[Sistema de Adopción y Gestión de Animales](https://github.com/sdahbaniiesruizgijon/refugio-animales.git)** 🐾 
   Plataforma web completa diseñada para la digitalización de protectoras y optimización de flujos de adopción. Cuenta con un sistema automatizado de notificaciones por correo electrónico (SMTP) que avisa en tiempo real a los usuarios sobre el estado de sus solicitudes, control de autenticación mediante Middlewares para proteger el Backoffice, y una interfaz moderna, limpia y 100% responsive maquetada con utilidades avanzadas de diseño.
   * *Tech Stack:* PHP (Laravel), Blade Templates, Tailwind CSS, MySQL (Migraciones y Seeders), SMTP Mail Service.
